# Cyber-Threat-Intelligence-Dashboard

A comprehensive dashboard for monitoring and analyzing cyber threats in real-time.

## Features

- Real-time threat monitoring
- Threat analysis and reporting
- Interactive dashboard visualization
- Security analytics

## Installation

1. Clone the repository:
```bash
git clone https://github.com/VijayKarthik12/Cyber-Threat-Intelligence-Dashboard.git
cd Cyber-Threat-Intelligence-Dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```

The dashboard will be available at `http://localhost:3000`

## Project Structure

- `index.html` - Main dashboard interface
- `server.js` - Backend server
- `package.json` - Project dependencies and scripts

## Technologies Used

- Node.js
- Express
- HTML/CSS/JavaScript
